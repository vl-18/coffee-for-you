# Coffee for You

## Project Documentation

### Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/vl-18/coffee-for-you.git
   ```
2. Navigate to the project directory:
   ```bash
   cd coffee-for-you
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the project:
   ```bash
   npm start
   ```

### Features
- User-friendly interface
- Search functionality for recipes
- Filter by caffeine levels
- Save favorite recipes

### Technology Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express
- MongoDB

### Coffee Recipes
1. **Espresso**
   - **Caffeine Level:** High
   - **Ingredients:** 18g coffee, 30ml water
   - **Instructions:** Brew under pressure.

2. **Americano**
   - **Caffeine Level:** Medium
   - **Ingredients:** Espresso, hot water
   - **Instructions:** Add water to espresso.

3. **Latte**
   - **Caffeine Level:** Medium
   - **Ingredients:** Espresso, steamed milk
   - **Instructions:** Mix together.

4. **Cappuccino**
   - **Caffeine Level:** Medium
   - **Ingredients:** Espresso, steamed milk, milk foam
   - **Instructions:** Layer in a cup.

5. **Mocha**
   - **Caffeine Level:** Medium
   - **Ingredients:** Espresso, steamed milk, chocolate
   - **Instructions:** Combine and stir.

6. **Macchiato**
   - **Caffeine Level:** High
   - **Ingredients:** Espresso, foamed milk
   - **Instructions:** Top espresso with foam.

7. **Flat White**
   - **Caffeine Level:** Medium
   - **Ingredients:** Espresso, microfoam
   - **Instructions:** Combine in a cup.

8. **Cold Brew**
   - **Caffeine Level:** High
   - **Ingredients:** Coarsely ground coffee, cold water
   - **Instructions:** Steep in cold water for 12-24 hours.

9. **Affogato**
   - **Caffeine Level:** Medium
   - **Ingredients:** Espresso, ice cream
   - **Instructions:** Pour espresso over ice cream.

10. **Nitro Cold Brew**
    - **Caffeine Level:** High
    - **Ingredients:** Cold brew coffee, nitrogen
    - **Instructions:** Infuse cold brew with nitrogen.

11. **Turkish Coffee**
    - **Caffeine Level:** Medium
    - **Ingredients:** Finely ground coffee, water
    - **Instructions:** Boil and serve unfiltered.

12. **Matcha Latte**
    - **Caffeine Level:** Medium
    - **Ingredients:** Matcha powder, hot water, milk
    - **Instructions:** Whisk matcha with water, then add milk.

### Matcha Recipes
1. **Traditional Matcha**
   - **Caffeine Level:** Low
   - **Ingredients:** Matcha powder, hot water
   - **Instructions:** Whisk until frothy.

2. **Iced Matcha Latte**
   - **Caffeine Level:** Medium
   - **Ingredients:** Matcha powder, milk, ice
   - **Instructions:** Mix matcha with milk over ice.

3. **Matcha Smoothie**
   - **Caffeine Level:** Low
   - **Ingredients:** Matcha powder, yogurt, banana
   - **Instructions:** Blend all ingredients.

4. **Matcha Cookies**
   - **Caffeine Level:** Low
   - **Ingredients:** Flour, sugar, matcha powder, butter
   - **Instructions:** Mix and bake at 350°F.

5. **Matcha Pancakes**
   - **Caffeine Level:** Medium
   - **Ingredients:** Flour, baking powder, matcha powder, eggs
   - **Instructions:** Mix and cook on skillet.

6. **Matcha Energy Balls**
   - **Caffeine Level:** Low
   - **Ingredients:** Oats, matcha powder, honey, nut butter
   - **Instructions:** Mix, roll into balls, and refrigerate.
